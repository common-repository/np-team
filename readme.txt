NP Team is an awesome plugin which will enable you to add team member option in your wordpress site.

== Description ==

NP Team is an awesome plugin for your WordPress website. This plugin will enable you to add team member option in your wordpress site with effects and plenty of options. You can change color & other setting from it's options panel.

It has a lot of options in option panel of this plugin. You can change Title color, Position Text Color, Description Color, Background color, Background Hover Color and Shadow Color from options panel of this plugin. Below you can see the list of the options name of the options panel of this plugin.

*Title color
*Position Text Color
*Description Color
*Background color    
*Background Hover Color
*Shadow Color

**How to use this plugin**
1. At first, you have to add members how much you want.
1. Then create a page and add team shortcode to the editor by clicking on NP-Team icon.


To see demo & documentation [click here.](http://nproject.byethost17.com/cv/np-team "Demo & Documentation")
